# PRos

Эмулятор 16-битной среды исполнения на базе движка Unicorn. Проект позволяет запускать низкоуровневый код (boot-секторы, ядра или простые программы), обеспечивая базовую поддержку видеовыхода VGA, клавиатурного ввода и системных прерываний.

## Функциональные возможности

*   **Процессор**: Эмуляция x86 в 16-битном режиме (Real Mode).
*   **Видео**: Поддержка текстового режима 80x25 и графического режима 13h (320x200, 256 цветов).
*   **Ввод**: Обработка клавиатуры через библиотеку minifb с трансляцией в скан-коды BIOS.
*   **Память**: Автоматическое определение адреса загрузки (0x0000, 0x7C00 или 0x8000) в зависимости от типа файла.

## Реализованные прерывания

### BIOS
*   **INT 10h**: Видеосервис (установка режима, управление курсором, вывод символов).
*   **INT 15h**: Системные функции (задержка usleep).
*   **INT 16h**: Работа с клавиатурой (чтение символа, проверка буфера).
*   **INT 1Ah**: Системное время (получение тиков таймера).

### DOS-like (Custom)
*   **INT 21h**: Вывод строк на экран, очистка экрана, установка цвета текста.
*   **INT 22h**: Файловая система (открытие, чтение, запись, листинг директории, удаление, переименование). Работает через системные вызовы хоста (libc).

## Системные требования

*   Rust
*   Библиотека Unicorn Engine
*   Зависимости: `minifb`, `libc`, `unicorn-engine`

## Сборка и запуск

1. Установите необходимые библиотеки в систему для сборки `unicorn-engine`.
2. Соберите и скачайте утилиту:
```bash
cargo install --path .
```
3. Запустите бинарный файл:
```bash
pros-emu файл.bin
```
